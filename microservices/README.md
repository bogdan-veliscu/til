# Python Microservices web app

## Tech Stack:
- Django Rest Framework
- Flask
- RabbitMQ
- MySql
- React


## Running locally

    -1 Install docker
    -1 cd to the project folder and start RabbitMq
    ```
    make run-mq
    ```
